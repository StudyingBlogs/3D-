# 3D大波浪
## css循序渐进
```
transition: all 1s;
```
## 移动
```
<!-- X,Y轴上的移动 -->
transform: translateX(300px);
```
## 动画animation
Transition属性配合鼠标操作，可以实现一些好看的过程，但是必须配合鼠标；animation属性就可以元素做一次或无数次的动画，不依赖于鼠标的操作；
```
animation: name duration timing-function delay iteration-count direction;
```
